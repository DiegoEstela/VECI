This is a technical test for VECI

To use this app, follow these steps: Click the “Add” button to add a new text element. You can select one or more text elements by clicking on them. Selected items will appear with a different background color. To delete a selected item, click the "Delete" button. If no item is selected, it will not delete anything. To undo the last action (add or delete), click the "Undo" button. If you want to add more text elements, repeat step 1. You can also close the modal without adding an element by clicking "Cancel" in the modal.
